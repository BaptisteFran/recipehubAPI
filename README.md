# RecipeHub API

Créer une API open-source de gestion de recettes permettant :
- stocker des recettes
- rechercher par ingrédients
- générer des listes de courses
- planifier des repas
- servir de backend pour différentes interfaces (web, mobile, CLI)

Le projet doit être :
- pédagogique (apprendre Python)
- modulaire
- testable
- documenté
- utilisable par d’autres projets

Python :
- types
- classes
- collections
- gestion erreurs
- manipulation données

Backend :
- API REST 
- validation
- architecture projet

Data :
- relations SQL
- requêtes
- agrégations

Qualité :
- tests unitaires
- organisation du code
- CI


### 🧭 Roadmap globale

#### Le projet est divisé en 8 milestones.

- 1️⃣ Initialisation du projet
- 2️⃣ API recettes (CRUD)
- 3️⃣ Gestion des ingrédients
- 4️⃣ Recherche avancée
- 5️⃣ Liste de courses
- 6️⃣ Planification de repas
- 7️⃣ Qualité et tests
- 8️⃣ Publication open source
